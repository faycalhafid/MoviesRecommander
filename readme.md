- Documentation test technique Moodwork.pdf : documentation qui explique la démarche, la méthodologie, réponses aux questions une par une, etc... Je vous conseille de commencer par ouvrir ce fichier, d'exécuter main.py, lire la réponse pour chaque question sur le PDF puis visualiser le résultat soit sur votre exécution ou sur le fichier Exemple d'execution.txt, puis lire la réponse de la question suivante, etc...
Les images présentes sur la doc sont aussi présentes en .png dans le dossier, pour pouvoir mieux visualiser et zoomer.

- main.py : code Python pour les calculs et les implémentations. Toutes les sections sont commentées, toutes les lignes sont commentées. Il suffit juste de l'exécuter, chaque action est accompagnée d'un print pour visualiser le résultat.

- Book1.twb : fichier de visualisation Tableau (les résultats sont sauvegardées sous forme d'images et figurent dans la documentation)

- Exemple d'execution.txt : fichier qui contient le résultat de tous les print présents dans le code Python, en réexécutant main.py sans y toucher, vous obtiendrez le même résultat
